Introduction
============

Overview
--------

Wellpath Analyst is a Python-based toolkit for the analysis, planning, and
visualization of directional well paths. It supports industry-standard
survey calculation methods and provides a modern, programmatic interface
for well trajectory workflows.

Key Capabilities
----------------

-  **Survey Import & Export** — Load survey data from CSV, Excel, WITSML,
   and LAS formats. Export to multiple standard formats.
-  **Trajectory Calculation** — Compute well path positions using Minimum
   Curvature, Radius of Curvature, Balanced Tangential, and Average Angle
   methods.
-  **3D Visualization** — Interactive 3D plots of single or multiple well
   paths with matplotlib and Plotly backends.
-  **Anti-Collision Analysis** — Separation factor and ellipsoid of
   uncertainty calculations for well proximity monitoring.
-  **Torque & Drag** — Simplified torque and drag modeling for tripping
   and drilling operations.
-  **Well Plan Design** — Build trajectory plans from target definitions
   using configurable BHA and well profile parameters.

Who Is This For?
----------------

Wellpath Analyst is intended for:

-  **Directional Drillers** planning and monitoring well trajectories.
-  **Drilling Engineers** performing anti-collision and well proximity
   checks.
-  **Geoscientists** integrating well path data into subsurface models.
-  **Software Developers** building drilling automation workflows.
